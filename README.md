# HomeServer Dashboard

This project serves as a centralized landing page for your home lab, providing quick links to various workspaces and services (e.g., Nextcloud, Plex, Home Assistant, AI server, Or jellyfin).

You can host this for the public Eye (Login page recomended) or at home, Login page also recomended
